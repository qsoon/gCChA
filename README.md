# gCChA

This repo contains the source code for CCA in the graph frequency domain.

## Description

- Code
  - `utils.R` is a code for functions used for gCChA.
  - `g20.R` is a code for real data application on G20 network data.
  - `usps.R` is a code for performing classification analysis on the USPS dataset.


- Data
  - `tradingnw.RData` contains the G20 trade network data for 2023.
  - `USPSdigits.RData` contains the USPS dataset from the IMIFA R package.
  - `BACI_HS92_V202501` (available at [http://www.cepii.fr/anglaisgraph/bdd/baci.htm](http://www.cepii.fr/anglaisgraph/bdd/baci.htm)), `economic`, and `energy` contain information on world trade, and economic and energy indicators.

## Code overview
We present a CCA method in the graph frequency domain.
